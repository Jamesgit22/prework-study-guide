HTML

  • The head element contains information about the webpage. This info is not viewable to the user in the browser.
  • (meta charset="UTF-8") describes what characters and symbols can be used on a webpage. 8 provides the most options.
  • The body element represents the visible content shown to the user.
  • The "a" tag stands for "anchor" and allows you to link other pages. (Use a href="url" to link external.
  • Only ONE h1 element should be used per webpage.
  • Margin, indicates how much space we want around the outside of an elemnent.
  • Padding, indicates how much space we want around the content inside an element.

GIT

  • git status: Checks what branch you are currently in.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git checkout main: Switches you to main branch

git flow to push new changes:

  1. Ensure you are on the correct branch using the (git status) command.
  2. Once verified, use (git add -A) to stage all files.
  3. Next, commit the changes using (git commit -m "your commit message").
  4. Before intiiating a pull request, ensure that your branch is up to date using (git pull origin main)
  5. Now push all the changes you have made to your branch to the main branch using (git push origin branch-name)
  6. Lastly, navigate to github to review the pull request and obtain any nessesary approvals before merging to the main branch.
